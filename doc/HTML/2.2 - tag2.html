语义标签

标题标签

标题标签，用于添加标题，HTML中一共有六个标题标签，<h1>-<h6>，根据重要性依次递减，这些标签都有默认加粗的效果。 

源码
<h1>一级标题</h1>
<h2>二级标题</h2>
......
<h6>六级标题</h6>
效果
一级标题
二级标题
......
六级标题

换行标签

与普通文本格式不同，HTML文本的换行并不会显示在页面中。

源码
文字a
文字b
效果
文字a文字b
这时就需要换行标签来进行换行 
源码
文字a<br>
文字b
效果
文字a
文字b

段落标签

段落标签可以让段落与段落之间换行，且加入一定间距，使文章结构更清晰。 

源码
<p>段落a</p>
<p>段落b</p>
效果
段落a
段落b

文本格式化标签

文本格式化标签可以为文本加入加粗斜体等样式，加强网页展示效果。

效果标签说明
加粗<strong></strong>或<b></b>这些标签虽然效果一样，但strong，em，del，ins语气更强烈，一般用于提醒或重点标注，具备特殊含义，而b，i，s，u只代表样式无特殊含义
斜体<em></em>或<i></i>
删除线<del></del>或<s></s>
下划线<ins></ins>或<u></u>
高亮<mark></mark>高亮显示文本
代码块<code></code>以等宽字体显示
上标2<sup></sup>上标，用于表示幂等
下标2<sub></sub>下标，用于表示化学式等
倒序<bdo dir="属性"></bdo>规定文字显示顺序，dir属性ltr即从左到右，rtl即从右到左基本没卵用

链接标签

路径

在了解链接标签前，需先了解路径有关内容。

相对路径

相对路径是相对当前文件的路径。
对于以下文件结构：

  • HTML
    • index.html
    • index2.html
    • en_US
      • index_en_US
  • images
    • pic1.webp
  • 404.html
此时以HTML/index为起点，index2.html文件的路径为index2.html,
/表示下一级路径，例如index/en_US/index_en_US.html
../表示上一级路径，例如../404.html或../images/pic1.webp

绝对路径

绝对路径指从某一根目录（如电脑盘符）开始或者某一网址的路径，例如： D:\web\images\pic1.webp或 

http://www.mywebsite.com/images/pic1.webp
由于电脑盘符路径取决于文件在电脑中的位置，当整个网站根文件夹的位置改变，或者运行在其他服务器上时，就完全无法运作，所以一般不会使用。
特别的，当网站运行在服务器上时，需选取一个文件夹作为网站的根目录，用户只能访问其中的文件，此时可以用如下方式表示网站的根目录: 
/images/pic1.webp

链接标签

链接标签用于定义超链接，从一个页面跳转到另一个页面。 

<a href="目标地址，可以是相对路径" target="标签页打开方式"></a>

属性

    target
  • _self：在当前标签页打开。
  • _blank：在新标签页打开。
